# Mediflow Web Aplication

## How to use it?

1. Create a virtual enviroment

```shell
python -m venv venv
```

2. Start the virtual enviroment

```shell
venv\Scripts\activate
```

3. Install the requirements .txt

``` shell
pip install -r ./requirements.txt
```

4. Start the server

``` shell
python manage.py runserver
```
